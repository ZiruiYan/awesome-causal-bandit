## What are causal bandit?
Causal Bandit is defined over a Probabilistic Causal Model $\mathcal{G} = (V,E)$ where $V$ is the set of vertices and $E$ is the set of edges.

### Reward
Nomally the last node $X_N$ is defined as reward.
